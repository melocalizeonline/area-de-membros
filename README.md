# Area de Membros

Plataforma de membros para liberar cursos, ferramentas e materiais conforme compras externas.

## Stack

- Next.js 16
- React 19
- Supabase Auth + Database
- Tailwind CSS 4

## Primeiros passos

1. Copie `.env.example` para `.env.local`.
2. Preencha as chaves do novo projeto Supabase.
3. Rode o schema em `supabase/schema.sql`.
4. Inicie o projeto com `npm run dev`.

## Modulos principais

- Membros
- Produtos
- Cursos
- Ferramentas
- Integracoes: Kiwify, Eduzz e provedores de video
